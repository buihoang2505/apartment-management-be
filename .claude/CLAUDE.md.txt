# Apartment Management — Project Brain

## Tổng quan
Hệ thống quản lý chung cư/bất động sản. Backend REST API viết bằng Java / Spring Boot.

## Tech Stack
- **Language:** Java 21+
- **Framework:** Spring Boot 3.x
- **Build:** Gradle multi-module
- **DB:** PostgreSQL (port 5433)
- **Auth:** JWT (Spring Security)
- **ORM:** Spring Data JPA + Hibernate
- **Utilities:** Lombok, Bean Validation

## Module Structure
apartment-management/
├── domain/        ← Entities, Repositories (JPA interfaces), domain logic
├── app/           ← CQRS Handlers (CommandHandler / QueryHandler), DTOs, exceptions
└── infrastructure/← Web layer (Controllers), Security (JWT), config
## Architecture: CQRS
- Mỗi feature có **CommandHandler** (write) và **QueryHandler** (read) riêng biệt
- Controller → Handler (không gọi Service trực tiếp)
- Domain entities KHÔNG phụ thuộc app/infrastructure

## Domain Entities
- `User` — người dùng hệ thống
- `Portfolio` — danh mục bất động sản
- `Zone` — khu vực / tòa nhà
- `Apartment` — căn hộ (có status, type, area, price)
- `AuditLog` — lịch sử thao tác (user, action, entityType, oldValue, newValue, ipAddress)

## Branching
- `main` — production
- `develop` — development (default working branch)

## Conventions
- Package: `com.apartment.<module>.<domain>.<layer>`
- IDs: `UUID`
- Base entity: `BaseEntity` (có createdAt, updatedAt)
- Exceptions: mỗi domain có exception riêng (vd: `PortfolioNotFoundException`)

## Frontend (./frontend)
- Vue 3 + Vite + Pinia + Vue Router + Axios + TailwindCSS
- Gọi API từ backend port 8080
- Thiết kế UI: xem docs/designs/
- Dark navy theme, bottom navigation, card layout

## Backend (./backend)
- Spring Boot port 8080
- Swagger: http://localhost:8080/swagger-ui/index.html

## Khi viết FE component
- Luôn dùng Composition API (không dùng Options API)
- Gọi API qua src/services/, không gọi axios trực tiếp trong component
- Tên component: PascalCase, tên file: kebab-case
