# Tech Defaults

## Java
- Version: 21+
- Luôn dùng `var` khi type rõ ràng
- Prefer `record` cho DTOs immutable

## Lombok
- `@Getter @Setter` trên Entity
- `@RequiredArgsConstructor` trên Handler/Controller
- `@SuperBuilder` cho Entity kế thừa BaseEntity

## JPA
- Fetch type mặc định: `LAZY`
- Dùng `@Column(name = "snake_case")` explicit
- UUID làm primary key

## Gradle
- Run: `./gradlew :infrastructure:bootRun`
- Test: `./gradlew test`
- Build: `./gradlew build`
