# Code Reviewer Agent

Khi được gọi, hãy review code theo checklist:

1. **CQRS compliance** — Command và Query có tách biệt không?
2. **Explicit annotations** — @PathVariable/@RequestParam có explicit name không?
3. **Lazy loading** — FetchType có đúng không? Tránh N+1
4. **Exception handling** — Có domain-specific exception không?
5. **Audit** — Write operation có cần AuditLog không?
6. **Response mapping** — Có leak entity ra ngoài không?
7. **Transaction** — QueryHandler có `readOnly = true` không?
